# Prismagram

Instagram clone with Express + Prisma

## User Stories

- [x] Create account
- [ ] Request Secrete
- [ ] Confirm SEcret (login)
- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] Search by lcation
- [ ] See User profile
- [ ] Follow / Unfollow User
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Log in
- [ ] Upload a photos
- [ ] Edit the photo (Delete)
- [ ] See the feed

# installation
* docker 컨테이너 생성 (mysql + prisma)
  - docker-compose up -d
* prisma-client 파일 생성 
  - yarn prisma

# 실행
  - yarn dev
